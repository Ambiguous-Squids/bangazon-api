##How to Run

Create a virtual environment
```
python -m venv venv
```
Activate the virtual environment
```
source venv/bin/activate
```
Install all dependencies in requirements.txt
```
pip install -r requirements.txt
```
To deactivate the virtual environment
```
deactivate
```
